About

This is the demo application of Webino the Open Source Website System.

This demo can't show all the features of Webino, but you have to tweak it!

This is the first alpha release, use it in production on your own risk. You can play with Webino to understand its principles.

This is the basic demo with text pages and pictures gallery. Works without database just with configuration. Static files like style images, cascading style sheets and java script are downloaded from the internet, so when you aren't online, demo wouldn't work.

Please visit project home page http://www.webino.org/
